## Guidelines

- **Format:** PNG, portrait, iPhone aspect ratio (e.g. 1290×2796 from an iPhone 15 Pro). GitHub scales them down in the README; no need to pre-shrink, but keep each file under ~1 MB.
- **Language:** capture in **English** to match the README copy. A per-language set is out of scope (see Epic #58 open points in [`../ROADMAP.md`](../ROADMAP.md)).
- **Privacy:** use demo/sample data — no real names, no exact location traces, no API keys on screen.
- **Consistency:** same device frame and light/dark mode across the set looks best.
